# To Do List With ReactJS


## Installation

```bash
$ git clone https://github.com/orcunakkaya/ToDoList.git
```

## Usage

```bash
 cd ToDoList
 npm install
 npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Features

 - Add task
 - Delete task
 - Mark task as done
 - Change Theme
 - Responsive Design
